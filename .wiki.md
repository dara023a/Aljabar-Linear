# Project Summary
The **SPL Calculator - Grab Cost Analysis** project is a modern web application designed to compute **Linear Equation Systems (SPL)** using the **Gauss** and **Gauss-Jordan** elimination methods. This tool assists users in analyzing transportation costs for Grab by allowing them to input various parameters and visualize results interactively, enhancing decision-making for transportation logistics.

# Project Module Description
The application consists of several key modules:
- **Input Module**: Allows users to enter journey data (distance, time, and cost) dynamically.
- **Calculation Module**: Implements Gauss and Gauss-Jordan methods to solve the equations and provides structured step-by-step solutions.
- **Visualization Module**: Displays results through interactive graphs comparing actual vs. predicted costs.

# Usage
### Prerequisites
- Node.js (version 18 or higher)
- pnpm (package manager)

### Installation Steps
1. Clone or download the repository.
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Run the development server:
   ```bash
   pnpm run dev
   ```

### How to Use
1. **Landing Page**: Click the **"Start Simulation"** button.
2. **Input Data**: 
   - Add new rows for journey data.
   - Remove rows as needed.
   - Load example data if desired.
3. **Calculate SPL**: Click **"Calculate SPL"** to see results.
4. **Visualize**: Click **"View Visualization"** to see graphs.
5. **Step-by-Step Display**: View detailed steps for Gauss and Gauss-Jordan methods in the results.
